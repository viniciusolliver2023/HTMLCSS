\ Imagens, áudios e vídeos!

Imagens adaptativas...

As imagens adaptativas servem para se adaptar a diferentes telas
de dispositivos sejam móveis ou não.

Vídeos de diferentes formatos...

Podemos colocar diferentes vídeos de acordo com a compatibilidade do
seu navegador.

Diferentes áudios...

No navegador, assim como os vídeos e imagens, podemos colocar
diferentes formatos de áudios de acordo com a compatibilidade.

Downloads...

Para despertar a função download é bem simples.
Só é necessário usar um plug-in no VScode chamado live server.
