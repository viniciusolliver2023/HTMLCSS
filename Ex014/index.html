CSS externos:

Em HTML há uma importância em usar o (código) HTML separado do estilo.
Isso se deve por vários fatores, mas o principal é: organização!

Você deve-se lembrar que na área de tecnologia as coisas podem se tornar
um pouco complicadas de serem manuseadas. Portanto, é compreensível que
haja uma forma de deixar tudo legitimo, tanto no entendimento do código por si, quanto
na organização visual do código (humanos lidando com máquinas).

Então a necessidade de separar o script de HTML do CSS nasceu
na necessidade de criar algo que ajudasse na legibilidade do código.
