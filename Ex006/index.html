Como criar um favicon...

Para criar um favicon, é necessário usar a tag <link>.
É uma tag de metadado, significa que funciona na área de configuração, o <head>.

A base do favicon é um atributo chamado rel que significa "relação".
O "rel" diz exatamente com o que aquela tag estará relacionada de acordo com o valor.
No caso, temos o valor de "shortcut icon", que significa que a tag link se comportará
como um favicon.

Logo após, temos o atributo href que é tipo um src, porém é usado para criar
uma interatividade ao usuário. É onde você colocará o diretório do arquivo .ico.

Após, temos o atributo type que dirá qual é o tipo de arquivo que será carregado.
O tipo funciona como Tipo/Subtipo. Se for uma imagem png será: image/png e se for
JPG será image/jpeg
