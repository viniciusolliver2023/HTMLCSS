Quebras

Um texto comum em HTML consome toda sua linha, independe se toda linha for preenchida ou não.
Isso é devido a natureza das tags. Uma tag pode ser inline ou block.

Uma tag inline funciona ocupando somente o conteúdo necessário na tela.
Isso significa que se houver uma outra tag inline logo após, ficará ao lado,
não abaixo.

Já uma tag block ocupa todo o espaço de uma linha. Por exemplo, a tag <p>
funciona como uma tag de block pois quando colocada após a outra, uma fica acima
e outra fica abaixo. Sendo assim, ela ocupa todo o espaço da tela.

O exemplo claro é a imagem que está no meio do texto. Em um caso comum, sem
parâmetros, o texto "vazaria" para fora da imagem, seguindo como se fosse um fluxo
de tag block, pois seu pai é a tag <p>.

A tag <img> é uma tag inline. Significa que ela ocupa somente o espaço
necessário na renderização do navegador. Porém, ao adicionarmos o parâmetro
align="left", ela torna um objeto flutuante na página. Isso significa
que ela sobrepõe as tags abaixo, se tornando um objeto a esquerda do navegador.
